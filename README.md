# ToothTyper
ToothTyper is a voice-to-text wrapper for Patterson Eaglesoft perio charting.
To use it, download and install it from the [releases](https://github.com/HexadecimalDevelopment/ToothTyper/releases/latest/). Then, open the perio charting window, open ToothTyper, and click "Start Listening". Ensure the perio window has focus. ToothTyper will minimize automatically. Wait until you hear a ding (ensure you system volume is up), and then you can begin dictating. To stop, click on the ToothTyper window and click "Stop Listening".
